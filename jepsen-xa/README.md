## Usage

### Development

Run the app in REPL.
```clojure
(def system
  (do
    (if (resolve 'system)
      (ig/halt! system))
    (ig/init 
      (load-config {:log/level {:app :debug :other :info}
                    :instrument true
                    :server {:port 3000
                             :join false}
                    :db {:db1 {:port 55432 :host "127.0.0.1"}
                         :db2 {:port 55433 :host "127.0.0.1"}}}))))
```
